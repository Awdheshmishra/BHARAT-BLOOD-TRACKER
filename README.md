# 🩸 Bharat Blood Tracker

> Real-time blood availability platform for Lucknow, UP — connecting patients, donors, and hospitals instantly.

![MongoDB](https://img.shields.io/badge/Database-MongoDB_Atlas-green)

## 🚀 Live Features
- ✅ Real-time blood inventory across Lucknow hospitals
- ✅ Nearest hospital finder using GPS
- ✅ Separate Hospital & User dashboards
- ✅ OTP-based mobile verification (Twilio)
- ✅ Blood request posting with live donor alerts
- ✅ Google Maps with color-coded markers
- ✅ Role-based JWT authentication

## 🛠️ Tech Stack
| Frontend | Backend | Database |
|----------|---------|----------|
| React 18 | Node.js | MongoDB Atlas |
| Vite | Express.js | Mongoose |
| React Router v6 | Socket.IO | GeoJSON 2dsphere |
| Context API | JWT Auth | TTL Index (OTP) |
| Google Maps API | Twilio SMS | |

## ⚡ Quick Start
```bash
npm run install:all
cd backend && node utils/seed.js && cd ..
npm run dev
```
